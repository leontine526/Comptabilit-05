Recommandations

  1. Vérifiez que toutes les routes non-publiques sont protégées par une authentification
  2. Ajoutez une validation des paramètres pour les routes qui en ont besoin
  3. Utilisez des transactions pour les opérations DB critiques

  6. Implémentez une gestion d'erreur robuste pour toutes les routes
  7. Ajoutez une protection CSRF pour les routes qui modifient des données
