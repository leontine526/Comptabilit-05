À propos de SmartOHADA

SmartOHADA est une application de gestion comptable basée sur le référentiel OHADA, conçue pour simplifier la comptabilité des entreprises dans les pays membres de l'OHADA.

Version SQLite

Cette version de l'application utilise SQLite comme base de données, ce qui la rend plus légère et facile à déployer. Elle comprend toutes les fonctionnalités essentielles pour la gestion comptable.
